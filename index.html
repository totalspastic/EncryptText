What is EncryptText ?

EncryptText is a website that allows the users to enter the any type of text and get it processed by our algorithms which results in it to change into cypher text; making the text unreadable by a third party.

The site is easy to use and is visually pleasing to the eye. EncryptText offers multiple options for the user to change their text into different formats.

Helps the user understand how the encryption occurs by providing the code for the corrosponding algorithm.

The current site lays down the foundation for the full website in which similar functionality will be present.

Benefits of using this site

  • Secures the entered data
  • Prevents any other party to look into your data
  • Helps the user better understand the use of encryption by giving a practical example
  • Repository for multiple algorithms that can be used for reference
